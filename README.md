# Zoho-Expense-Tracking-Framework
Advanced expense tracking system using Zoho Books, including categorization, owner reimbursement, and billable expense management.

---

## 📌 Project Overview

The goal of this project was to:
- Record and categorize multiple travel-related expenses  
- Set up a proper liability account for reimbursements  
- Ensure visibility through dashboards and reports  
- Apply correct accounting treatment for billable expenses

## 🧾 Chart of Accounts Setup

A dedicated liability account was created to properly track reimbursements:

- **Account Name:** Due to Owner (Owner Reimbursement)  
- **Account Type:** Other Current Liability  
- **Structure:** Standalone account (not a sub-account)  
- **Purpose:** Tracks funds owed to Jim for out-of-pocket expenses  

This ensured accurate financial reporting and clear visibility on the Balance Sheet.
